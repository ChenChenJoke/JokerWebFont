参数说明：

offset: Number 默认选中项的偏移量（默认值1为第二项，应修改为0）
navSize: Number 一屏呈现的项数（默认值3，应修改为1）
itemSize: Number 每一项的像素宽度
scrollDuration: Number 滚动过程花费的时间（默认为300毫秒）
autoScroll: Boolean 是否自动滚动（默认值false）
autoDuration: Number 两次滚动的时间间隔（默认值5000毫秒）
containerId: String 轮播容器的选择器
dir: String 页面的文字流（默认值为"ltr"，可选"rtl"）
direction: String 滚动的轴向（默认值为"h"横向滚动，可选"v"表示纵向滚动）
autoScrollDirection: String 自动滚动发生的方向（默认值"forward"表示前向或向下滚动，可选项"backforward"表示向后或向上滚动）
data: Array 数组的每一项是一个包含属性"content"和idKey的key-value对象（"content"是该项的HTML字符串，idKey是该项的标识值）
idKey: String 每一项的唯一标识符（默认值为"id"）
defaultId: String 第一项的idKey值
quickSwitch: Boolean 快捷切换的控制开关（默认值false，可选项true表示开启快捷切换）
showTitle: Boolean 为每项添加title属性的开关（默认值为false，可选项true将为每一项添加title属性值为其内容本身）
